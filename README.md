# BB app

Werkstructuur voor de nieuwe BB/AEK app op Astro + Client-First + Supabase.

## Structuur

- `apps/web` bevat het daadwerkelijke project dat we gaan bouwen
- `docs` bevat plannen en technische notities
- `references` bevat oud naslagwerk en frameworkreferenties

## Referenties

- `references/client-first-framework` is de Client-First basis
- `references/notive-app-scrap` is het oude Notive/Beter Boompje naslagwerk

## Startpunt

De eerstvolgende bouwstappen landen in `apps/web`.
